# FrameUnlocker

Unlock and customize your UI frames. Drag chat by its tab, resize it from the corner, and scale raid frames to your preference.

## Features

### Chat Frame
- Unlock chat to drag by tab and resize from corner
- Stays clamped to screen so it can't be lost off-edge
- Automatically re-unlocks after exiting Edit Mode
- Toggle on/off in settings

### Raid Frames
- Scale compact raid frames from 50% to 150%
- Adjustable via slider in settings

### Settings Panel
- Full options UI integrated into the AddOns menu
- All preferences persist between sessions
- Reset to defaults button
- Slash commands reference

## Commands

| Command | Description |
|---------|-------------|
| `/fu` | Open settings panel |
| `/fu unlock` | Force unlock chat |
| `/fu lock` | Lock chat |
| `/fu scale` | Apply raid frame scale |
| `/fu reset` | Reset all settings to defaults |
